# Sistema de información Editorial Shalom

## ¿Cómo montarlo localmente?

### Requerimientos
- Node JS
- PostgreSQL
- PGAdmin
- VSCode
- Postman

### Backend
Servidor hecho en Node JS + Express.

```
cd Frontend
npm install 
```
```
Desarrollador: npm run dev  
Producción: npm build 
```

### Frontend
Parte visual hecha en Vue JS 3.

```
cd Frontend
npm install 
```
```
Desarrollador: npm run dev  
Producción: npm build 
```
